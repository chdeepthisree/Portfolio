Disease detection and Dietary suggestions using machine Learning

Technologies Used: Python,Machine Learning

Created a python based machine learning model which focuses on disease prediction based on user inputted symptoms.
We have implemented an effective way to reduce the dimensionality,eliminating the irrelevant data and increasing the accuracy.
It analyzes the input symptoms and predicts the most probable disease based on the patterns it has learned during training. If the user selects a predicted disease, the system retrieves information about dietary recommendations specific to that disease. This information may include dietary restrictions and general nutritional guidelines to manage the condition effectively.
Along with dietary recommendations, the system provides a description of the predicted disease.
The system is presented through a user-friendly interface, allowing users to easily input symptoms,view predicted diseases, and access additional information about specific diseases and dietary recommendations.
